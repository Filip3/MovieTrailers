# MovieTrailers

###### Description of the sample

The soultion implements a basic search of movie trailers, which uses API from two sources:

- IMDB
- YouTube

###### Prerequisites

This sample requires:
 
- Visual Studio


###### To start the sample

- Clone the project
- Restore nuget packages 
- Build the sample
- Run and test the sample on lochost
